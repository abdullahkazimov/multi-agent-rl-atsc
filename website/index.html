Methodology

Thesis §3 — Training protocol, reward design, and evaluation procedure.

🤖

PPO with Shared Policy

Proximal Policy Optimization via Stable-Baselines3. A single MLP policy (128×128) controls all TLs in a scenario via a MultiDiscrete action space — one green-phase selection per TL per 10-second decision step.

🚦

7-Term Composite Reward

Reward balances five objectives with thesis-calibrated β weights: throughput (β=0.15), stopped-vehicle ratio (β=0.30), waiting time (β=0.55, primary), queue length (β=0.12), and phase-starvation penalty (β=1.00).

📊

Matched-Snapshot Evaluation

For each seed, the identical demand schedule is run under both fixed-time and PPO control. 1 000 decision steps per episode, first 100 discarded as warm-up. Δm = (FT−RL)/FT×100% for minimisation metrics.

About This Project

Thesis

This website presents the experimental results from an MS thesis on multi-agent reinforcement learning for adaptive traffic signal control, conducted at ADA University's School of Information Technologies and Engineering.

All four SUMO scenarios are modelled on real Baku, Azerbaijan road networks and calibrated to realistic demand levels. The codebase is built on Stable-Baselines3, SUMO 1.18, and Gymnasium 1.2.
